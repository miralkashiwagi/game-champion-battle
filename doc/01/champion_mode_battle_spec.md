# LostSAGA — チャンピオンモード 戦闘仕様書

**バージョン:** v1.1  
**対象:** 開発チーム向け実装仕様  
**対象範囲:** キャラ選択 / キャラ登録 / 戦闘フロー / アイテム / 勝敗判定  
**前提:** キャラクター共通仕様（`common_character_spec.md`）を参照のこと

---

## 1. 試合全体フロー

```
キャラ選択（ロビー入室時）
  ↓
キャラ登録（選択したキャラで登録）
  ↓
マッチング（CPが近い相手と）
  ↓
戦闘フェーズ（1v1）
  ↓
勝敗確定（どちらかが死亡）
  ↓
CP更新
```

---

## 2. キャラ選択画面

| 項目 | 内容 |
|------|------|
| 表示タイミング | チャンピオンモードのロビー入室時 |
| 選択可能キャラ | プレイヤーが所持しているキャラのみ表示 |
| 選択数 | 1体のみ |
| 初期実装キャラ | シルバーナイトのみ（今後追加予定） |
| キャラ追加方針 | 新キャラは各キャラ仕様書（`[キャラ名]_spec.md`）を作成の上、選択画面に順次追加する |
| 選択後の変更 | マッチング開始前であれば何度でも変更可 |
| 選択画面のUI要件 | キャラ名・アイコン・タイプ（近距離 / 遠距離 など）を一覧表示。詳細確認ボタンでスキル概要を参照可能 |

---

## 3. キャラ（用兵）登録

| 項目 | 内容 |
|------|------|
| 登録タイミング | 試合参加前にロビー画面で事前登録 |
| 登録可能キャラ数 | 1体のみ |
| 登録変更 | 試合開始前であれば何度でも変更可。試合開始後は変更不可 |
| 装備持ち込み | 登録時に装備しているギアセットをそのまま持ち込む |
| 試合中の用兵交代 | 不可（通常モードとの差異点） |

---

## 4. 戦闘フェーズ

| 項目 | 内容 |
|------|------|
| 形式 | 1v1 |
| 勝利条件 | 相手プレイヤーを死亡させる（1Kill制） |
| 使用ステージ | 専用ステージ1種類 |

---

## 5. ダメージ進行モデル

```
通常状態（HP あり）
  ↓ HP が 0 になる
アイテム剥ぎ取りフェーズ
  被弾ごとに装備を1枚ずつドロップ（マント → 頭 → 胴 → 武器）
  ↓ 全4枚ドロップ後にさらに被弾
死亡（試合終了）
```

詳細は共通仕様（`common_character_spec.md` §1〜§3）を参照。

---

## 6. フィールド上アイテムと拾い直し

| 項目 | 内容 |
|------|------|
| ドロップ後の挙動 | 剥ぎ取られたアイテムはフィールド上にドロップされる |
| 消滅タイミング | 試合終了まで消滅しない |
| 拾得権 | 自分・相手どちらでも拾得可能 |
| 自分のアイテムを拾った場合 | 装備スロットに戻り、対応するスキルが復活。HPは回復しない |
| 相手のアイテムを拾った場合 | スワップ発生（詳細は共通仕様 §2-3 参照） |

---

## 7. 勝敗判定

| ケース | 判定 | 判定基準 |
|--------|------|---------|
| 相手が死亡した | 勝利 | — |
| 自分が死亡した | 敗北 | — |
| 制限時間切れ | HP比較 → アイテム数比較 → 引き分け | ① 残HPが高い方の勝利<br>② 両者HP=0の場合、装備中のアイテム数が多い方の勝利<br>③ アイテム数も同数の場合、引き分け |
| 同時死亡 | 引き分け | — |

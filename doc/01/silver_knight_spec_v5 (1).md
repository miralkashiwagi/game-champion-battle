## 2. 基本攻撃・コマンド

### 2-1. 通常コンボ（D→D→D→D）

**武器スロットに含まれる。武器を失うと素手状態になり使用不可。**

| 段 | モーション時間 | ヒット効果 |
|----|---------------|-----------|
| 1段目 | 0.1秒 | — |
| 2段目 | 0.1秒 | — |
| 3段目 | 0.1秒 | — |
| 4段目 | 0.1秒 | **浮かせダウン** → 空中ダメージ状態へ移行 |

- 各段ごとに次の入力を受け付ける
- 4段目ヒット後は追撃可能

---

### 2-2. 突き（Dホールド）

**武器スロットに含まれる。武器を失うと使用不可。**

| フェーズ | 時間 |
|----------|------|
| 溜めモーション | 0.3秒 |
| 突き発生 | — |
| 硬直 | 0.3秒 |

- ヒット効果: **膝つきダウン**（共通仕様 §4-1 参照）

---

### 2-3. ガード

共通仕様（`common_character_spec.md` §6-1）に準拠。

---

### 2-4. ガードカウンター

| フェーズ | 時間 | 内容 |
|----------|------|------|
| ガード成立後の猶予 | 0.5秒 | 共通仕様に準拠 |
| 専用モーション | 0.1秒 | 斬り上げ → 相手が空中に浮く（空中ダメージ状態へ移行） |
| 硬直 | 0.1秒 | 復帰 |

- ガード成立後 0.5秒以内に攻撃ボタンで発動
- ヒット効果: 相手を浮かせる → 通常コンボへの追撃が可能

**ガードカウンター後のコンボ例:**

```
ガードカウンター → [浮かせ] → D→D→D→D → [浮かせダウン]
```

## 3. 装備スキル

### 3-1. マント — 体当たり ※オリジナル仕様

**種別:** アクティブ / 攻撃・移動  
**クールタイム:** なし  
**試合開始時:** 即使用可

| フェーズ | 時間 | 内容 |
|----------|------|------|
| 予備動作 | 0.4秒 | 身体をかがませて突進姿勢（相手に読まれるリスク） |
| 突進 | — | 大きく前進（距離は通常移動の **3倍**） |
| ヒット | — | 相手を上空へ浮かせ → 空中ダメージ状態へ移行 |
| 硬直 | 0.3秒 | 復帰 |

**代表コンボ例:**

```
体当たり → [浮かせ] → D→D→D→D → [浮かせダウン]
```

**備考:**
- 予備動作中に相手に読まれると回避・反撃されるリスクがある
- 空振り時は突進距離が大きいため大幅に位置がずれる
- CTなしのため連発可能。ただし予備動作リスクとのトレードオフ

---

### 3-2. 頭 — ヘッドバッド

**種別:** アクティブ / 攻撃  
**クールタイム:** 5秒  
**試合開始時:** 即使用可

| フェーズ | 時間 | 内容 |
|----------|------|------|
| 溜め | 0.3秒 | 行動制限あり |
| 頭突き発生 | — | リーチ短め・発生早め |
| ヒット | — | **気絶（約3秒）** |
| 硬直 | 0.1秒 | 復帰 |
| クールタイム | 5秒 | 使用後 |

**気絶の仕様:**
- 気絶中は相手が操作不能 → 追撃・コンボが狙いやすい
- 気絶中に攻撃を受けると気絶が解除される（共通仕様 §4-4 参照）

---

### 3-3. 胴 — ハードガード

**種別:** パッシブ / 防御  
**クールタイム:** なし  
**試合開始時:** 常時有効

- 近接攻撃を被弾した際、相手を自動で一定距離押し返す
- ゲージ消費・入力不要
- 発動条件: **立ち状態のみ**（空中・浮かされ中は無効）
- のけぞり（0.2秒）発生と同時に押し返しが発動
- 一部の遠距離攻撃（見た目が近接に見えるもの）には無効

---

### 3-4. 武器 — スラッシュ

**種別:** アクティブ / 攻撃  
**クールタイム:** 20秒  
**試合開始時:** CT中（即使用不可）  
**ガード貫通:** あり

| フェーズ | 内容 |
|----------|------|
| 発生 | 即時（予備動作なし） |
| 地上版 | 3段斬り。3段目: 吹き飛びダウン → 空中ダメージ状態へ移行 |
| 空中版 | 単発急降下。着地後のスキ小 |
| 無敵時間 | あり（切り返しとして機能） |
| ガード貫通 | **相手がガード中でも当たる** |
| クールタイム | 20秒 |

**備考:**
- 試合開始時はCT中のため序盤は使用不可
- ガード貫通は武器スキルのみの特権（共通仕様 §6-3 参照）
- 20秒のCTにより乱用不可。使いどころが勝負の鍵

## 5. 装備喪失時の影響

| 失った装備 | 失うスキル | 戦略的影響 |
|-----------|-----------|-----------|
| マント | 体当たり（コンボ始動） | コンボ始動手段を失う |
| 頭 | ヘッドバッド（気絶） | 気絶からの追撃が不可能に |
| 胴 | ハードガード（押し返しパッシブ） | 防御力低下・攻め込まれやすくなる |
| 武器 | スラッシュ（ガード貫通・無敵）+ 通常コンボ（D×4）+ 突き（Dホールド） | ガード貫通手段・通常攻撃手段をすべて失い素手状態になる。次の被弾で死亡 |

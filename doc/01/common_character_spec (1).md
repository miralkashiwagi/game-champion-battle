# キャラクター共通仕様書

**バージョン:** v1.1  
**対象:** 開発チーム向け実装仕様  
**適用範囲:** 全キャラクター共通

---

## 1. 装備スロット構成

全キャラクターは以下の4スロットで構成される。

| スロット | 部位 | 剥ぎ取り順 |
|----------|------|-----------|
| マント | 背部 | 1番目（最初に落ちる） |
| 頭 | 頭部 | 2番目 |
| 胴 | 胴体 | 3番目 |
| 武器 | 手持ち | 4番目（最後に落ちる） |

- 剥ぎ取り順は固定（マント → 頭 → 胴 → 武器）。ランダムではない
- **武器スロットには「スキル」と「通常攻撃モーション」の両方が含まれる**
- 武器を失うと通常攻撃モーションも失われ、素手状態になる（§1-1 参照）

### 1-1. 素手状態

武器スロットを失った場合に適用される共通の攻撃仕様。

| 項目 | 内容 |
|------|------|
| コンボ段数 | 3段（D→D→D） |
| 1段あたりのモーション時間 | 0.2秒 |
| 威力 | 武器装備時より低め |
| 武器スキル | 使用不可 |

- 素手状態は全キャラ共通。キャラ固有の通常攻撃モーションは武器に含まれる
- 次の被弾で死亡するため、素手状態での戦闘継続は極めて困難

## 2. アイテム・スキル移転仕様

### 2-1. ドロップとフィールド上の挙動

- HPが0になった後、被弾するたびに装備が剥ぎ取り順に従って1枚ずつフィールドにドロップされる
- ドロップしたアイテムは試合終了まで消滅しない
- 自分・相手どちらでも拾得可能

### 2-2. 自分のアイテムを拾った場合

- 装備スロットに戻り、対応するスキルが復活する
- HPは回復しない

### 2-3. 相手のアイテムを拾った場合（スワップ）

- 自分がその部位に装備していたアイテムが代わりにフィールドへドロップされる（入れ替わり）
- 拾った装備のスキルが使用可能になる
- 元々持っていたスキルは使用不可になる

**スワップ例:**

```
相手のマントがフィールドに落ちている状態で自分が拾う
  → 自分のマントスロットのアイテムがフィールドへ落ちる
  → 自分は相手のマントスキルを使用可能に
  → 相手は自分のマントスキルを拾い直せる状態になる
```

## 3. ダメージ進行モデル

```
通常状態（HP あり）
  ↓ HP が 0 になる
アイテム剥ぎ取りフェーズ（被弾ごとに装備を1枚ずつドロップ）
  ↓ 全4枚ドロップ後にさらに被弾
死亡（試合終了）
```

## 4. 状態定義

### 4-1. 膝つきダウン状態

特定の攻撃ヒット時に発生（各キャラのスキル仕様を参照）。

| フェーズ | 時間 | 操作 |
|----------|------|------|
| 膝つき | 0.5秒 | 操作不能 |
| 倒れ込み | 1.0秒 | 操作不能 |
| ダウン状態へ移行 | — | §4-3 参照 |

- 膝つき中・倒れ込み中に攻撃を受けると **空中ダメージ状態** に移行する

---

### 4-2. 空中ダメージ状態

以下のいずれかで発生する。

- 空中で攻撃を受けた
- 膝つきダウン中に攻撃を受けた
- 浮かせ効果を持つ攻撃にヒットした

| フェーズ | 条件 | 操作 |
|----------|------|------|
| 空中ダメージ中 | 地面に着くまで | 操作不能 |
| 着地 | 地面に設置した瞬間 | **ダウン状態** へ移行 |

---

### 4-3. ダウン状態

空中ダメージ状態から着地すると移行。

**攻撃判定:**
- 通常攻撃: **当たらない**
- ダウン攻撃判定を持つ技のみ有効（各キャラの仕様を参照）

**起き上がり操作:**

| 入力 | 動作 | 無敵 | 備考 |
|------|------|------|------|
| 前進 or 後退 | その方向へ回転しながら起き上がり | 回転中は無敵 | — |
| 攻撃ボタン | その場で起き上がりながら攻撃 | 攻撃中は無敵 | 攻撃後硬直 0.4秒。ガードされると反撃リスクあり |

---

### 4-4. 気絶状態

特定の攻撃ヒット時に発生（各キャラのスキル仕様を参照）。

- 気絶中は操作不能
- **気絶中に攻撃を受けると気絶が解除される**
- 解除後は通常状態に戻る

## 5. 被撃・のけぞり仕様

### 5-1. 地上 被撃

| フェーズ | 時間 |
|----------|------|
| のけぞりモーション | 0.2秒 |
| 行動可能 | のけぞり終了後 |

- のけぞり中は行動不能
- 連続被弾時はのけぞりがリセットされる

### 5-2. 空中 被撃

- 被弾した瞬間から **空中ダメージ状態** に移行（§4-2 参照）
- 地面に着くまで操作不能

## 6. ガード・カウンター仕様

### 6-1. ガード

| 項目 | 内容 |
|------|------|
| 有効方向 | 前方からの攻撃のみ |
| 最大継続時間 | 3秒（押し続けても3秒で強制解除） |
| 解除後 | 再入力で再ガード可能 |

### 6-2. ガードカウンター

| フェーズ | 内容 |
|----------|------|
| ガード成立 | — |
| カウンター入力猶予 | 0.5秒以内に攻撃ボタン |
| 専用モーション | 発動（キャラごとに異なる） |
| 猶予超過時 | カウンター不可。通常ガード継続（3秒上限に含まれる） |

- ガードカウンターの専用モーション・フレーム数は各キャラの仕様を参照

### 6-3. ガード貫通

- 一部の武器スキルはガード中の相手にも当たる（ガード貫通）
- ガード貫通の有無は各キャラのスキル仕様を参照
